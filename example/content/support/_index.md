## Quick Fixes

### Common Issues

**Navigation not showing:**
- Check `config.yaml` menu configuration
- Verify content files exist in correct directories
- Clear Hugo cache (`hugo --gc`)

**Styling not applying:**
- Ensure theme is correctly installed
- Check CSS custom properties
- Verify file paths in config

**Build errors:**
- Update Hugo to latest version
- Check front matter syntax
- Validate TOML configuration
